# Computadores e Programação - CP DEF da FCTUC

## Instalação

Você pode clonar este repositório (descarregar os arquivos) utilizando Git ou pode baixar os arquivos diretamente pelo Github.

Para utilizar o Git é preciso instalar esta ferramenta no seu computador.Acesse git-scm.com e realize as etapas de instalação. 

### Se tiver GIT

Com o git instalado, abra um terminal dentro da pasta desejada e execute o comando:

```
git clone https://github.com/ProspektsMarch22/computadores_e_programacao_DEFUC.git
```

### Se não tiver GIT

Pressione o botão "<> code" e, sem seguida, "Download ZIP".

Descompacte os arquivos e os mova para uma pasta desejada.

## Utilização

Todos os exercícios resolvidos contém um main(), e portanto podem ser tratados como executáveis.

Dessa forma, para conferir o programa funcionando, basta utilizar o comando

```
python3 <nome_do_programa>
```

Você **deve** fazer isso na pasta onde o programa está. Fora da pasta o comando não roda.
